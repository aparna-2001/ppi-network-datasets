DATABASE: DIP (Database of Interacting Proteins)
================================================================================

Version        : Current release
Organism       : Homo sapiens (Human)
Taxonomy ID    : 9606
Download URL   : https://dip.doe-mbi.ucla.edu/dip/Download.cgi
Date Documented: (fill in today's date)

NOTE: DIP requires free registration for download.
      Register at: https://dip.doe-mbi.ucla.edu/dip/Register.cgi

FILES TO DOWNLOAD
-----------------
1. Hsapi.txt.gz    → Human protein interactions (tab-delimited)
2. Hsapi.mif25.gz  → PSI-MI XML format (alternative)

FILE FORMAT
-----------
ID interactor A | ID interactor B | Alt IDs A | Alt IDs B |
Aliases A | Aliases B | Detection method | Author |
PubMed ID | Taxon A | Taxon B | Interaction type |
Source DB | Interaction ID | Confidence

EVIDENCE TYPES INCLUDED
------------------------
- Experimentally determined only
- Yeast two-hybrid (Y2H)
- Co-immunoprecipitation
- Biochemical assays
- Each interaction linked to PubMed reference

APPROX. SIZE
------------
Nodes (proteins)   : ~5,000
Edges (interactions): ~25,000

LICENSE   : Free for academic and non-commercial use
CITATION  : Salwinski et al., "The Database of Interacting Proteins: 2004 update",
            Nucleic Acids Research, 2004.
            DOI: 10.1093/nar/gkh086

NOTES
-----
- One of the oldest and most established PPI databases
- Smaller dataset but very high reliability
- Experimental evidence only — no predictions
- Requires free registration to download
- Maintained by UCLA
