DATABASE: MINT (Molecular INTeraction database)
================================================================================

Version        : Current release
Organism       : Homo sapiens (Human)
Taxonomy ID    : 9606
Download URL   : https://mint.bio.uniroma2.it/download
Date Documented: 05/05/2026

FILES TO DOWNLOAD
-----------------
1. mint.zip     → All interactions in MITAB format
2. mint.txt     → Tab-delimited full dataset

FILE FORMAT
-----------
MITAB 2.7 format (same structure as IntAct)
- Interactor IDs, detection method, publication reference,
  interaction type, confidence score

EVIDENCE TYPES INCLUDED
------------------------
- Manually curated from scientific literature
- Experimental interactions only
- Literature-based evidence with PubMed references

APPROX. SIZE
------------
Nodes (proteins)   : ~10,000
Edges (interactions): ~300,000

LICENSE   : Creative Commons CC BY 4.0
CITATION  : Licata et al., "MINT, the molecular interaction database: 2012 update",
            Nucleic Acids Research, 2012.
            DOI: 10.1093/nar/gkr930

NOTES
-----
- Developed at University of Rome Tor Vergata
- Focuses on manually curated, high-quality interactions
- Smaller but highly reliable dataset
- Data is also integrated into IMEx consortium
- Compatible with IntAct data format
